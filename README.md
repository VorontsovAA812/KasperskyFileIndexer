# File Indexing Service (Kaspersky Lab Test Task)

Библиотека и консольное приложение для полнотекстового поиска по локальным текстовым файлам с поддержкой многопоточности и отслеживания изменений в реальном времени.

## Особенности реализации

* **Многопоточность:** Индексация файлов распараллелена с помощью `ExecutorService` (Fixed Thread Pool). Количество потоков адаптивно под количество ядер процессора.
* **Динамическое обновление (WatchService):** Мониторинг файловой системы через `Java NIO`. Индекс обновляется автоматически при добавлении или изменении файлов.
* **Расширяемость:** Механизм токенизации вынесен в интерфейс `Tokenizer`, что позволяет легко заменять логику разбиения текста.
* **Thread-safe:** Хранилище индекса на базе `ConcurrentHashMap` обеспечивает безопасный одновременный доступ.

## Требования

* **Java:** 17 или выше
* **Maven:** 3.6+

## Инструкция по сборке и запуску

### 0. Клонирование репозитория
Склонируйте проект на свой компьютер и перейдите в папку с проектом:
```bash
git clone https://github.com/VorontsovAA812/KasperskyFileIndexer.git
cd KasperskyFileIndexer
```
### 1. Сборка проекта
Выполните сборку (это также запустит все тесты):
```bash
mvn clean package
```
После сборки в папке `target/` будет создан исполняемый файл `KasperskyFileIndexer-1.0-SNAPSHOT.jar`.

### 2. Запуск приложения
Запустите программу, указав путь к директории для индексации:
```bash
java -jar target/KasperskyFileIndexer-1.0-SNAPSHOT.jar /путь/к/папке
```

### 3. Запуск тестов
Для проверки модульных тестов выполните:
```bash
mvn test
```

## Структура проекта
* `api/` — интерфейс Tokenizer.
* `impl/`  — конкретная реализация токенизатора
* `model/` — потокобезопасное хранилище индекса.
* `service/` — сканирование и мониторинг изменений.
* `exception/` — обработка ошибок индексации.
